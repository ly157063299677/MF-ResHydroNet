# Data Directory

This repository does not include raw hydrodynamic simulation data.

Use:

```text
data/demo/
```

for synthetic demo data generated by `scripts/make_demo_data.py`.

Use:

```text
data/manuscript/reference_tables/
```

for IEEE Access manuscript-reference summary tables exported by
`scripts/build_reference_tables.py`.

Use:

```text
local_data/
```

for private raw or derived data when rerunning the full study locally. The
`local_data/` folder is ignored by Git.
